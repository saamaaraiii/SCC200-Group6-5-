# SCC200 Group 6-5 — AI Suggestions & Contributions

This repo contains AI-generated suggestions, improvements, and additions for the [SCC200 Group 6-5 project](https://github.com/saamaaraiii/SCC200-Group6-5-).

Nothing here modifies the original repo directly — everything is a suggestion or addition that the team can review and pull in if they find it useful.

## Structure

| Folder | Contents |
|--------|----------|
| `suggestions/` | Task lists, ideas, and recommendations generated during heartbeat checks |
| `notes/` | Code analysis, explanations, and context notes on existing files |
| `readme-improvements/` | Suggested README files for components that are missing them |
| `code/` | New code additions (connectors, utilities, helpers) the team can adopt |

## How this works

An AI assistant monitors the group project repo periodically and:
- Reviews new commits and changed files
- Suggests tasks that look unfinished or could improve the project
- Writes README files for components that don't have them
- Adds utility code (e.g. connecting APIs, data pipelines)
- Flags anything worth the team's attention

All suggestions are timestamped and labelled by what triggered them.
