# SCC200-Group6-5-

Group 6-5 for group project at Lancaster University.

## Python Backend

- Entry point: `backend/main.py`
- Install dependencies: `python -m pip install -r requirements.txt`
- Run server: `python backend/main.py`
- Run tests: `python -m unittest -v backend/test/test_api_python.py`

Detailed backend endpoint documentation is in `backend/README.md`.
